## Overview

You can use cross-references to establish directional relationships between classes. To define a cross-reference, you must:
- Set up [`cross-reference` property](../config-refs/datatypes.md#datatype-cross-reference) in one class (`from`) schema referring to another (`to`) class, and
- Establish the cross-reference between a `from` class object to a `to` class object.

:::note Cross-references do not affect vectors
Creating cross-references do not affect object vectors in either direction.
:::
